# sankum.in (static)

This is a multi-page HTML/CSS/JS portfolio site with shared header/footer (partials) loaded via a tiny JS include.

## Local preview
Use VS Code + Live Server, or run:

```bash
python -m http.server 5500
```

Then open http://localhost:5500

## Deploy (GitHub Pages)
Settings → Pages → Deploy from a branch → main → /(root)

## Custom domain
Add `sankum.in` in Settings → Pages → Custom domain and enable HTTPS.
